# Week 2 Assignments - ATP

This folder contains assignments for **Week 2**, focusing on advanced JavaScript topics, ES6 features, and an introduction to Node.js backend development.

## Directory Structure

### [Assignments(DAY3)](./Assignments(DAY3)/)
- **ArrayAdvancedOperations**: Exercises using advanced array methods (`map`, `filter`, `reduce`) on arrays of objects.
- **Modules**: Implementation of modular code using ES6 modules.
  - `E-commerce`: A modular approach to a simple e-commerce logic.
  - `Modular-todo`: A todo list application split into modules.

### [Assignments(DAY4)](./Assignments(DAY4)/)
- **Spreadoperator & restParameter**: Exercises demonstrating the use of spread and rest operators.
- **TimerFunctions**: Working with `setTimeout` and `setInterval`.
- `classes.js`: Introduction to JavaScript Classes and Object-Oriented Programming (OOP) concepts.

### [Assignments(DAY 5)](./Assignments(DAY%205)/)
- **backend**: Initial steps into Node.js backend development.
  - `server.js`: Basic Express/Node.js server setup.
  - `package.json`: Project dependencies and scripts.
  - `req.http`: Sample HTTP requests for testing the server.

## Concepts Covered
- Advanced Array Methods
- ES6 Modules (Import/Export)
- Spread and Rest Operators
- Asynchronous JavaScript (Timers)
- Classes and Inheritance
- Basic Node.js & Express Setup
