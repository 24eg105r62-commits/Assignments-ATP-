# Week 3 Assignments - ATP

This folder contains assignments for **Week 3**, focusing on backend development with Node.js, Express, and organizing code using the MVC pattern.

## Directory Structure

### [Assignments(DAY7)](./Assignments(DAY7)/)
- **product backend**: A robust backend implementation for managing products.
  - **APIs**: Route handlers for product endpoints.
  - **model**: Data models for product information.
  - `server.js`: Main entry point for the Express server.

## Concepts Covered
- RESTful API Design
- Express Routing
- Model-View-Controller (MVC) Architecture
- Middleware implementation
