# Week 3 Assignments - ATP

This folder contains assignments for **Week 3**, focusing on backend development with Node.js, Express, and organizing code using the MVC (Model-View-Controller) pattern.

## Directory Structure

### [Assignments(DAY7)](./Assignments(DAY7)/)
- **product backend**: A robust backend implementation for managing products.
  - **APIs**: Route handlers and logic for product-related endpoints.
  - **model**: Data models and schemas for product information.
  - `server.js`: The main entry point for the Express server.
  - `req.http`: HTTP test files for API validation.

## Concepts Covered
- RESTful API Design
- Express Routing
- Model-View-Controller (MVC) Architecture
- Middleware implementation
- Data Persistence (Mock or Database)
