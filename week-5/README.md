# Week 5 Assignments - ATP

This folder contains assignments for **Week 5**, covering advanced CSS techniques (Flexbox and Tables) and an introduction to modern frontend frameworks and libraries like React and Tailwind CSS.

## Directory Structure

### [Assignments(DAY-12)](./Assignments(DAY-12)/)
- **flex**: Exercises focusing on CSS Flexbox for responsive and flexible layouts.
  - `1.html` & `1.css`: Practical application of Flexbox properties.
- **tables**: Working with HTML tables and CSS styling for data representation.
  - `table1.html` - `table3.html`: Different table structures and styling levels.

### [Assignments(DAY-13)](./Assignments(DAY-13)/)
- **React**: Introduction to React.js, component-based architecture, and state management.
- **Tailwind-css**: Using utility-first CSS for rapid UI development.

## Concepts Covered
- CSS Flexbox (Justify-content, Align-items, Flex-direction)
- Advanced HTML Tables & CSS Styling
- Utility-First CSS with Tailwind
- React Basics (Components, Props, JSX)
