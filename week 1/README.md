# Week 1 Assignments - ATP

This repository contains the JavaScript assignments for **Week 1** of the Advanced Technical Programming (ATP) course.

## Directory Structure

### [Assignments(DAY1)](./Assignments(DAY1)/)
Focuses on basic logic and array processing.
- `array_min.js`: Finds the minimum value in an array.
- `array_sum.js`: Calculates the sum of elements in an array.
- `biggest_three.js`: Finds the largest of three numbers.
- `biggest_two.js`: Finds the larger of two numbers.
- `function_arrSum.js`: Function-based approach to sum array elements.
- `function_bignum.js`: Function to find the biggest number.
- `function_target.js`: Searches for a target value in an array using a function.

### [Assignments(DAY2)](./Assignments(DAY2)/)
Focuses on more advanced array operations and object manipulation.
- `Array_operations.js`: Demonstrates CRUD-like operations on an array of employee objects.

## Concepts Covered
- Variables and Data Types
- Conditional Statements
- Loops (`for`, `for...of`)
- Functions and Parameters
- Array Methods (`splice`, `length`)
