# Week 6 Assignments - ATP

This folder contains assignments for **Week 6**, focusing on building full-stack applications and advanced React concepts like State Management, Context API, and CRUD operations.

## Directory Structure

### [addingusers](./addingusers/)
- A React application focused on user management.
- Demonstrates how to handle forms and dynamic lists in React.

### [editCounter](./editCounter/)
- A state management project featuring multiple counter components.
- Uses **Context API** to share and edit state across different parts of the application.
- Includes `EditCounter1` through `EditCounter4` to show different implementation patterns.

### [employee](./employee/)
- A full-stack employee management system.
  - **[backend](./employee/backend/)**: Node.js/Express server with MongoDB integration.
  - **[frontend](./employee/frontend/)**: React-based dashboard for employee operations (Create, Read, Update, Delete).

### [react-app-2](./react-app-2/)
- A comprehensive React application demonstrating advanced UI patterns and component composition.

## Concepts Covered
- Full-Stack Development (MERN Stack Basics)
- Advanced React Hooks (`useContext`, `useState`, `useEffect`)
- Context API for Global State Management
- REST API Integration with React
- Form Handling and Validation
- Responsive UI Design with Modern CSS
